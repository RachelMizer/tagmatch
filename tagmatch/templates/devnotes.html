Requirements & Dependencies

Core

  • Python 3.13
  • Django 5.2.8
  • SQLite — default dev database (PostgreSQL recommended for production)
  • asgiref 3.11.0 — ASGI support layer for Django
  • sqlparse 0.5.3 — SQL formatting (Django dependency)

Installed Packages

  • django-taggit 6.1.0 — tag management for user profiles and matching
  • django-filter 25.2 — search and filtering support
  • Pillow 12.0.0 — image handling: profile photo upload, EXIF correction, square crop, resize
  • django-storages 1.14.6 — pluggable file storage backends (AWS S3 ready)
  • boto3 1.41.3 / botocore 1.41.3 — AWS SDK (required by django-storages for S3)
  • tzdata 2025.2 — timezone data
  • python-dateutil 2.9.0 — extended date/time parsing

Local Development Setup

  • Virtual environment (.venv)
  • .env file at project root — holds SECRET_KEY and DEBUG (loaded automatically by settings.py, never committed to git)
  • Run: python manage.py migrate before first launch
  • Run: python manage.py runserver to start dev server at http://127.0.0.1:8000
  • Static files: python manage.py collectstatic (production only)

Recommended Tools

  • VS Code
  • Git + GitHub (github.com/RachelMizer/tagmatch)
